# Our plan:
Calculating carbon footprint based on a few daily actions: transportation, meat consumption, water usage, purchases etc; offer suggestions for improvement (based on inputs)

# Some computational tasks
1) Use rock paper scissors algorithm for giving choices (car, airplane, bike, walking, invalid)
2) Use carbon footprint data for each activity, calculate based on user inputs
3) Find a list of suggestions using if statements, or dictionary to select proper recommendations


# FP Status Update:
# Description
Our project now prompts a user to think about and input various metric in their daily routine that contribute to their carbon footprint: transportation, meat consumption, and water usage. Based on a user's habits, the program then generates a few suggestions tailored to help the individual reduce their carbon footprint and negative environmental impact.

# User Instructions
1) Run updatedFPstatus.py in your terminal
2) As prompted, enter your mode of transportations (car, bike, etc.)
3) As prompted, enter your normal distance traveled (consider things like commute, daily bike rides, etc.)
4) As prompted, enter your average shower length in minutes
5) As prompted, enter an estimate of the number of burgers/servings of red meat you consume weekly
6) Consider the suggestions offered by the program!
